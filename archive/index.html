# 《我的大学：怀念在校的八个月》

为什么只有八个月呢？

  • 2019年9，10，11，12 大一
  • 2020年9，10，11，12 大二

2019年9月刚入校，大一上刚结束。这不一个疫情，大一下就一直在家上网课。2020年9月来到学校，大二上稍微时间长一点儿四个多月吧，暂且按四个月计。

为什么要写blog？主要是我喜欢删pyq。今晚基本上算是最后一次用电脑了，发个blog熟悉下git的操作。

# 大一上

大一上啊！我就是个憨批。回顾在校园的四个月，学习方面似乎就是个混子。对了，这个时候我还不是个工科生，我是个管科生。那个啥吧，一天就混得一批，感觉也没学什么吧。唯一能记得的两门课就是：《大学语文》和《管理学原理》。

大一我在学校都在干嘛？我就自己“瞎搞”。这几个月吧，我感觉我成长的很快，人也很浮躁，这是要分好几个层面讲的。

从相册翻到以前的照片还是很想回忆一下的。
